VMware

	ctrl+alt+INS   = soft reboot, quickly see BIOS options
			 F2  = enter bios
			 F12 = network boot (default)
			 ESC = boot menu

	Easiest thing is to use pre-defined image for CDROM.
	If not, soft reboot using ctrl+alt+ins will NOT disconnnect client CDROM
	so that helps get things booted up for setup.

	-t 1 = create vmdk of up to 2GB, add files as needed (growth on demand)
	ubuntu32-data is the prefix used for the vmdk filename.
