AIX CD Catalog

AIX does not have a "install everything" option. So, one often has to dig thru CDs to find the right package to add a desired feature. Here is what is in some of the CD. The vary by releases, so pretty long list. But easier to search here than keep on swapping CDs :) 


AIX 5.1L cd (5 cd set 10/2002)

CD 1 - bos.adt, termcap, perftools, X11, 
CD 3 - Java 1.3.0, X11.Dt, ldap, printers, 
cd 4,5,6 seems to be localization related cds.



AIX 5.2L cd installation.  7 cd set from 05/2004.

CD 1 - cifs, termcap info, bos.net, bos.perf, perfagent.tools.   (def: java 1.3.1)
       dos utilities, fonts for hp laster jet printer
CD 2 - CDE, fonts
cd 3 - some ldap, ldap_db2 stuff (2003 version with 6 cd set)
cd 5,6,7 seems to be localization related cds (lang, OS msg file map).

cd1 has bos.perf, which allow installation of bos.perf.perftools 
a package that provides Sys V /proc tools similar to solaris proctree, 

http://www-106.ibm.com/developerworks/eserver/articles/AIX5.2PerfTools.html
