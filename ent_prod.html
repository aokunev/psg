Enterprise Products Overview

Here is a list of various enterprise products used to archive high availability commonly used with a Sun centric shop.

Host Layer

  • Sun OS: IP Network Multipathing
    Redundancy by using multiple NIC and IP, also provides better performance for outbound traffic. Included with standard Sun OE.

  • Host Clustering Layer

  • Sun Cluster. SunPlex Manager provide web GUI and CLI.
  • Veritas Cluster (VCS)

    Storage, access to disks

  • Sun StorEdge Traffic Manager
  • Sun Cluster Disk Path Monitoring (DPM) w/ command scdpm
  • Veritas Dynamic Multi Path (DMP)
  • EMC PowerPath (PP)

    Storage, disk volume management and file system.

  • Sun Volume Manager (SVM), formerly Solstice Disk Suite (SDS).
  • Veritas Volume Manager (VxVM)
  • Sun Unix File System (UFS)
  • Sun StorEdge QFS
  • Veritas File System (VxFS)

    Storage management

  • Sun RAID Manager 6 (RM6) for A1000 disk array.
  • CLI for Sun T3 array.
  • EMC Navisphere (Web GUI) for Clariion.
