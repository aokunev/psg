(article in linux.com)

Linux performance monitoring and tracing tools
  • top, ntop, htop
  • mpstat
  • sar
  • free, swapon, cat /proc/meminfo
  • pidstat (sysstat rpm)
  • iotop (iostat rpm)
  • perf (linux-tools-generic rpm, + kernel specific rpm) , eg:
    perf top perf stat -a # hit ^C to end collection of stat and view result.
  • slaptop
  • fio (fio rpm)
  • sysbench (sysbench rpm) eg cmd:
    sysbench --test=cpu run
    sysbench --test=fileio --file-test-mode=seqrewr run
  • stap (systemtap rpm) need some script...
  • lttng (lttng-tools rpm)
  • dtrace (systemtap-sdt-dev rpm)



advanced tools to perform full software stack analysis such as
  • systemtap
